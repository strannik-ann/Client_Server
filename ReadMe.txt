Сборка:
	В папке _build запустить cmake ..
	make all
Запуск:
	./Server_main ./Client_main в той же папке
	Клиент обрабатывает 1 запрос
	Сервер останавливается по команде "exit" или "quit"
Комментарии по коду:
	Protocol - общая для клиента и сервера библиотека работы с протоколом TCP
	Адрес и порт задаются в Protocol.h
	
	В программе используется алгоритм Евклида для подсчета НОД. НОК = a * b / НОД
